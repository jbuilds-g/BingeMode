# BingeMode 🍿

> **Track your series. Zero fluff.**

> BingeMode is a lightweight, privacy-focused TV show tracker. It runs entirely in your browser using local storage, with optional TMDB integration for rich metadata.

---

## 🚀 Features
- **Smart Tracking:** Connect TMDB API (API key required) to get automatic season/episode counts and covers.
- **Manual Mode:** Track shows simply by season and episode number without any external data.
- **Progress Visuals:** See exactly where you are in a season with visual progress bars.
- **Offline Ready:** Installs as a PWA and works without an internet connection.
- **Data Freedom:** Export your data to JSON and import it anywhere.
  
---

## 🛠 Tech Stack
- Vanilla JavaScript
- IndexedDB
- CSS Grid / Variables
